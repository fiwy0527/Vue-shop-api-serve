## 结合 Vue-shop 项目使用 api 接口

## 使用方法

> > npm install 安装响应依赖
> > 使用 mysql 运行 db 下的 sql 文件
> > 同时需要修改对应 config 文件下的 ueditor.config 里面对应数据库的个人参数

    "db_config" : {
    	"protocol" : "mysql",
    	"host" : "127.0.0.1",
    	"database" : "mydb",
    	"user" : "root",
    	"password" : "",
    	"port" : 3306
    }
